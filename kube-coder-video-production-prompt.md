### 2. WORKSPACE FEATURES TABLE
- **Dashboard SPA** - Vite + Preact app
- **Terminal** - ttyd-attached tmux
- **VS Code** - code-server at `/home/dev`
- **In-pod browser** - Chrome on X display via noVNC
- **Metrics + health** - Live CPU/Mem/Disk monitoring
- **Assistant** - Claude Code, OpenCode, or Ante
- **Auth** - GitHub OAuth2 integration
